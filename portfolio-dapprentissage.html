Réaliser un développement d'application

Concevoir, coder, tester et intégrer une solution informatique pour un client

AC 1.1 : Élaborer et implémenter les spécifications fonctionnelles et non fonctionnelles à partir des exigences

CE : En respectant les principes d'accessibilité et d'ergonomie

Jeu de Société Train - Analyse et spécifications

Trace : Code source, documentation technique
J'ai travaillé cette compétence lors du développement du jeu Train en analysant minutieusement les règles du jeu original pour les transformer en spécifications techniques. J'ai élaboré l'architecture MVC, défini les interfaces utilisateur avec drag & drop, et spécifié les fonctionnalités (sauvegarde, multijoueur, tutoriel). Les spécifications non-fonctionnelles incluaient les performances, la fluidité des animations et l'ergonomie de l'interface JavaFX.

Stage Chaire Score - Application d'analyse IA

Trace : Application web, rapport de stage
Durant mon stage, j'ai analysé les besoins métier pour automatiser l'analyse de rapports financiers. J'ai spécifié une interface permettant de modifier les questions d'analyse, défini l'architecture FastAPI/LangChain, et implémenté des fonctionnalités d'archivage et d'historique. L'accessibilité était cruciale car l'application devait être utilisée par des chercheurs non-techniques.
Niveau de maîtrise :
Je maîtrise cette compétence grâce à ma capacité à transformer des besoins complexes en spécifications techniques claires, démontrée sur des projets variés (jeu, application IA, sites web).

Optimiser des applications informatiques

Analyser et optimiser des applications pour assurer les performances attendues

AC 2.1 : Choisir des structures de données complexes adaptées au problème

CE : En utilisant des techniques algorithmiques adaptées

Jeu Train - Structures de données optimisées

Trace : Code Java, algorithmes de pathfinding
J'ai implémenté des structures de données complexes pour gérer l'état du jeu : graphes pour représenter les routes ferroviaires, algorithmes de pathfinding optimisés pour calculer les meilleurs trajets, et structures efficaces pour la gestion des tours de jeu. Ces choix ont permis d'optimiser les performances et la fluidité du gameplay.

Application IA - Optimisation des requêtes LLM

Trace : Code Python, architecture LangChain
Pour l'application d'analyse IA, j'ai optimisé la gestion des données en structurant efficacement les questions d'analyse, en implémentant un système de cache pour éviter les requêtes redondantes au LLM, et en organisant les données de façon à minimiser les temps de traitement des rapports volumineux.
Niveau de maîtrise :
Je maîtrise cette compétence en sachant analyser les problèmes complexes et choisir les structures de données les plus adaptées pour optimiser les performances.

AC 2.2 : Utiliser des techniques algorithmiques adaptées pour des problèmes complexes

CE : En évaluant l'impact environnemental et sociétal

Code Game Jam - Optimisation temps réel

Trace : Code Unreal Engine, optimisations graphiques
Lors de la Game Jam, j'ai appliqué des techniques d'optimisation avancées sous Unreal Engine 5 : gestion efficace des ressources 3D, optimisation des rendus graphiques pour maintenir un framerate stable, et algorithmes de culling pour améliorer les performances. J'ai aussi considéré l'impact énergétique en optimisant la consommation GPU.

Nuit de l'Info - Optimisation web écologique

Pour le site sur le climat, j'ai appliqué des principes d'éco-conception : optimisation du JavaScript, réduction des temps de chargement, compression des images, et design léger pour minimiser l'empreinte carbone. Cette approche était cohérente avec le message environnemental du projet.
Niveau de maîtrise :
Je maîtrise cette compétence en intégrant les considérations de performance et d'impact environnemental dans mes choix algorithmiques et techniques.
